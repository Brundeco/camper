Een andere manier om het picture element te gebruiken is door verschillende groottes aan de hand van hoeveel plaats ze zullen innemen op de website.
Hiervoor geef je bij srcset de breedte (in pixels) van je afbeelding mee ipv de resolutie; In onderstaand voorbeeld is xlarge.jpg 1024px breed, dus we geven dit mee in de srcset.

srcset="img/xlarge.jpg 1024w
Het volgende attribuut dat we moeten invullen is "sizes". Deze vertelt de browser hoeveel pixels hij nodig heeft door te berekenen hoe breed de gerenderde afbeelding moet zijn.
Deze kan je ingeven als een vaste of relatieve waarde, px, ems, %, ... sizes="33.3vw" Je kan ook een breakpoint meegeven wanneer deze size veranderd, adhv een mediaquerie
Je laatste waarde is de default size sizes="(min-width: 36em) 33.3vw, 100vw"
